Quoifeur's odyssey
Quoifeur's Odyssey est un site tournant autour du jeu Super Mario Odyssey, je l'ai crée dans le but d'avoir tout une documentation sur une pratique: le speedrun. Une pratique consistant à terminer un jeu le plus rapidement possible, plus d'information concernant cette pratique sont disponible directement sur le site. Ce site à été crée dans le but de pouvoir avoir accès a certaines données à n'importe quel moment. C'est à l'heure actuelle le site le plus sophistiqué que j'ai pu réaliser. Aussi réalisé dans le but d'écrire du HTML, CSS car j'apprécie faire la chose, ce site comptes quelques secrets et mystères cachés.
