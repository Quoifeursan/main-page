Jouer à Pixelmon sur bedrock

1- Télécharger le fichier .zip ici, puis extraire le contenu

2- Cliquer sur le fichier PokeMod 1.0.0.mcaddon afin de le lancer avec Minecraft



3- Si tout a bien fonctionné, en haut une bandrolle affichant que l'importation est en cours



4- Cliquez sur jouer, puis appuyer sur le boutton "importer" à droite de "créer nouveau"





5- Sélectionner la map "PixelMon Map.mcworld"



6- Ajoutez le mod sur la map, cliquez sur le bouton à droite pour modifier la map



7- Descendez tout en bas sur le menu à gauche, dans "pack de comportement"



8- Cliquez sur "activés" afin de voir si le pack est déjà activés



9- Si vous ne voyez pas le pack, allez en dessous dans "mes pack" cliquez sur le pack et "activer"



Voilà, Vérifiez bien d'être en survie avant de lancer la map sinon vous ne pourrez pas lancer de pokemon :)
